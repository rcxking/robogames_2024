# robomagellan_localization

This package contains files and code related to helping the robot localize
itself.

## Launch Files

The following is a list of the Python launch files and their purpose:

1. `odometry_motion_model.launch.py`: Spawns the odometry motion model for a
   planar robot.  This model is from Probabilistic Robotics, Chapter 5.

## Nodes

The following is a list of Nodes within this package:

1. `odometry_motion_model.hpp/.cpp`: Implementation of the odometry motion
   model.
